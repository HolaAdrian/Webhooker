This plugin can be downlaoded here: https://www.spigotmc.org/resources/discord-webhooker-config.113168/

With this plugin you can send messages to your discord by using webhook.
It has an config so you can enable/disable the different events.

-> Events:
--> Player Join/Quit Event
---> Custom Title and Message
--> Player Death Event
---> Custom Title and Message
